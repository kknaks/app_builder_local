- **Docker Compose 자동 생성 서비스** (`app/services/docker_service.py`)

- **앱 실행/중지 API** (`app/routers/docker.py`)

  - `GET /api/projects/{id}/run/status` — 컨테이너 상태 + URL

- **통일된 에러 핸들링** (`app/core/error_handlers.py`)

  - ValidationError → 422 + 필드별 상세 에러
  - 미처리 예외 → 500 + 로깅
  - WS 연결 끊김 graceful 처리
- **E2E 통합 테스트** (`tests/test_e2e_flow.py`)
  - 전체 플로우: 생성→기획→검토→승인→스프린트→구현→Docker실행

  - 상태 전이 검증 (잘못된 전이 시 400)
  - Docker 라이프사이클: run→status→stop→re-run
